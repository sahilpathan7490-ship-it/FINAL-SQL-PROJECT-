# Final SQL Project – Student Management System

## Project Overview
This project is a MySQL-based Student Management System designed to demonstrate database design, data manipulation, and data analysis using SQL.

The database manages:
- Students
- Courses
- Instructors
- Departments
- Enrollments

The project includes:
- Database creation
- Table creation with Primary Keys and Foreign Keys
- Sample data insertion
- CRUD operations (Create, Read, Update, Delete)
- Aggregate functions
- Joins
- Subqueries
- Filtering and grouping queries

## Sample Tasks Performed

1. CRUD operations on all tables.
2. Retrieve students enrolled after a specific date.
3. Display courses offered by a specific department.
4. Count students enrolled in each course.
5. Find students enrolled in multiple courses.
6. Use DISTINCT to remove duplicate values.
7. Calculate averages using aggregate functions.
8. Apply GROUP BY and HAVING clauses.
9. Use subqueries for complex filtering.
10. Maintain referential integrity using foreign keys.
